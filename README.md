# Inventory Management System (Python)

This project is a simple **Inventory Management System** built using Python. It allows users to manage products in an inventory using a command-line interface.

## Features

* Add new products
* View all products
* Search product by ID
* Update product quantity
* Save and load data from a file

## Technologies

* Python
* Object-Oriented Programming (OOP)
* JSON for storing data

## Project Structure

```
product.py
inventory_manager.py
main.py
products.json
```

## Run the Project

Run the program using:

```
python main.py
```

## Purpose

This project demonstrates basic Python concepts such as classes, lists, file handling, and building a simple inventory system.
